# Single Grain Website Content Guide

This document contains all the production-ready website copy for Single Grain, structured for easy integration into components.

## Content Structure

All content is in `src/lib/sg-final-content.json`. This JSON file contains:

- **home**: Homepage content (hero, sections, CTAs)
- **services**: All service pages (SEO, Paid Media, Content, CRO) + overview
- **aiGrowth**: AI Growth / Search Everywhere page
- **caseStudies**: Case studies page + 3 example studies
- **about**: About page content
- **contact**: Contact page content
- **careers**: Careers page content
- **microcopy**: Reusable CTAs, trust bullets, footer content

## How to Use

1. Import the JSON file in your components
2. Map sections to your component structure
3. Use the SEO metadata for each page
4. Use microcopy for buttons, CTAs, and trust signals

## Content Principles

- **Tone**: Premium, direct, modern, operator mindset
- **Voice**: AI-forward, ROI-focused, expert-led
- **Style**: Short paragraphs, scannable, bullet points where helpful
- **Proof**: Metrics labeled as "typical outcomes" or "reported by clients"
